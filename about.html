About This Project

I have used Blender to create 3D models for this site. To expand on the marking criteria, I have also created a glass bottle model, utilizing a fingerprint image texture on top of a material that simulates glass.

The SQLite3 database is used to store data to be shown in the gallery carousel. A Flask script is used to serve the contents of the database to the carousel. The database can be initialised via the a simple python script, included in the project. The database can be exported in either XML or JSON format, using the download buttons at the bottom of this page.

The viewer utilizes Three.js to render 3D models in the browser. Bootstrap is used to style lighting controls within the viewer (three-viewer.js) and the models are loaded depending on the accordion selection above the viewer. Additionally, the background can be set by the user, allowing them to upload images.

The viewer utilizes Three.js to apply names to the glass bottle's label, done by writing text to a canvas that has the same colour as the rest of the label, then applying it as a texture to a specific part of the model's geometry.

Besides the Three.js viewer, this site uses JavaScript to handle the gallery carousel on the homepage and to manage the cookies. There is a small game using JavaScript and jQuery (with jQuery used to manage user input) where the user can attempt to throw a can into a recycling bin.

The website has a dark/light theme toggle, which is saved in a cookie. This allows the user to return to the site and have their theme preference saved.

All dependencies are obtained through CDN URLs, including Bootstrap, Bootstrap Icons, and Three.js. The project also uses a custom script (scripts.js) to handle the gallery carousel and other interactive elements.
